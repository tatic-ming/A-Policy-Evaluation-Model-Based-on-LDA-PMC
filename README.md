# A-Policy-Evaluation-Model-Based-on-LDA-PMC
This code builds and implements a technique that involves cleaning policy texts, analyzing complex policy social networks, performing LDA topic analysis, evaluating the consistency of policy texts (using the PMC scoring framework), and synthesizing PMC scores for policy texts across different regions using a quantile scoring model. It implements a text analysis tool for computing infrastructure-related texts across regions and time periods, based on NLP processing.
